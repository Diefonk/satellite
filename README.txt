Satellite v0.3
by Diefonk

Instructions:
CRANK to select synth
LEFT and RIGHT to select value
UP and DOWN to change selected value on selected synth (interval can only be changed when synth is muted)
A to toggle mute on selected synth
B to toggle mute on all synths (if any synth is not muted all will be muted, otherwise all will be unmuted)

All data (except mute status) is saved and loaded automatically (only one save for now)
